# Vue商城后台管理系统

## 项目介绍
- 该项目是一个商城后台管理系统的前端项目，基于Vue+Element-ui实现。主要包括用户管理、权限管理、商品管理、订单管理、数据管理、设置等功能。

## 项目演示
- 在线演示地址：https://monster-crj.github.io/management-system/
